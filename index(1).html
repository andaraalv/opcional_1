Ejemplo de uso del elemento blockquote

El elemento blockquote se usa para citar textos largos de una fuente externa. A continuación, un ejemplo:

La imaginación es más importante que el conocimiento. El conocimiento es limitado, la imaginación rodea al mundo. – Albert Einstein

El ejemplo anterior cita una frase famosa de Albert Einstein y muestra cómo se puede usar el atributo cite para referenciar la fuente.
